Vytvoríme jednoduchú hororovú hru pre jedného hráča. Jeho cieľom bude násjť východ
z neznámeho prostredia, pričom je naháňaný príšerami a má obmedzené informácie
o mape (pohľad 2D zhora, vidí len to, čo je pred ním, čo je blízko). Mapa bude
náhodne generovaná pri spustení levelu a zabezpečí tak vždy nový zážitok,
a to aj tvorcom hry.

Činnosti                                    Odhadované časy
generovanie levelu na začiatku              20 čh
jednoduchá tiles grafika + pohyby spritami  5 čh
príšery (+AI)                               10 čh
atmosferické hororové efekty                10 čh
help / jednoduché herné UI                  2 čh

Jazyk: Python 
Platforma : Linux (Windows), spúšťanie ako skript .py
Vývojové prostredie : väčšina PyCharm
Testovanie: python unittest, štandartný python framework, aj podporovaný v PyCharme
Úložisko: github
Integrácia: master brach + reviewované pull requesty
